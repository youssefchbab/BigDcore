\begin{abstract}
\paragraph{}
The RISC-V architecture has seen a significant rise in popularity due to its energy efficiency and modular design compared to traditional Complex Instruction Set Computer (CISC) architectures like x86. This project focuses on the design and implementation of a pipelined RISC-V processor that fully supports the RV32I Base Integer Instruction Set. The processor was developed using VHDL as the hardware description language, and ModelSim was utilized as the primary tool for functional simulation and verification. The design process involved the creation of a dedicated datapath, control unit, memory interface, and a hazard unit to manage pipeline stages. The results of the simulation confirm that the processor successfully executes the standard RV32I instruction suite with full functional correctness. This project demonstrates the feasibility of custom RISC-V implementations for low-power embedded systems and provides a modular foundation for future extensions, such as the privileged architecture.
